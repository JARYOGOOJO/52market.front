1. 카카오톡 소셜 로그인

* 로그인/로그아웃/회원가입
* 기가입자 이메일 일치시 갱신  

2.게시글 뉴스피드

* 🚩다양한 게시글을 보여줌
* 🚩무한스크롤 구현  
* 🚩SPA 페이지 구현  
* 🚩내 주변 이웃들(1km 이내) 글만 보기
* 🚩게시글 업로드 시 다른 사용자에게 알람 기능

3.댓글 기능

* 🚩게시글에 댓글 작성 가능
* 🚩댓글의 버튼을 통해 웹소켓 실시간 채팅 기능
* 🚩웹소켓 이용해 댓글 추가 및 삭제 시 즉시 반영
